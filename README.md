## Project Description
This project implements a banking application and tests it using PyTest. It includes unit tests for individual functions and integration tests for combined workflows like fund transfer and interest calculation.

## How to Run Tests
1. Install dependencies:
pip install -r requirements.txt

2. Run all tests:
pytest -v

3. Generate HTML Report:
pytest --html=report.html -v

## GitHub Actions
GitHub Actions is used to automatically run unit and integration tests whenever code is pushed to the repository.
